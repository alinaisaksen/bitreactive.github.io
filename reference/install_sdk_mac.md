<h2><a name="install_the_java_development_kit" id="install_the_java_development_kit">Install the Java Development Kit</a></h2>
<div class="level2">

<p>

Since working with Reactive Blocks means doing some Java development, you need a full Java Development Kit (<acronym title="Java Development Kit">JDK</acronym>) installed (not just a JRE). 

</p>
<ul>
<li class="level1"><div class="li"> Install Java directly from Apple, as <a href="http://support.apple.com/kb/DL1421" class="urlextern" title="http://support.apple.com/kb/DL1421"  rel="nofollow">explained here</a></div>
</li>
</ul>

</div>
<!-- SECTION "Install the Java Development Kit" [47-335] -->
<h2><a name="get_eclipse" id="get_eclipse">Get Eclipse</a></h2>
<div class="level2">

<p>

Download Eclipse 4.3.1 (Kepler) from <a href="http://www.eclipse.org/downloads" class="urlextern" title="http://www.eclipse.org/downloads"  rel="nofollow">here</a>.

</p>
<ul>
<li class="level1"><div class="li">     Choose the Eclipse <acronym title="Software Development Kit">SDK</acronym></div>
</li>
<li class="level1"><div class="li">     Choose the COCOA version</div>
</li>
<li class="level1"><div class="li">     If in doubt, install the 32-bit version</div>
</li>
<li class="level1"><div class="li">     More information on installing Eclipse is available <a href="http://wiki.eclipse.org/Eclipse/Installation" class="urlextern" title="http://wiki.eclipse.org/Eclipse/Installation"  rel="nofollow">here</a></div>
</li>
</ul>

</div>
<!-- SECTION "Get Eclipse" [336-668] -->
<h2><a name="install_eclipse" id="install_eclipse">Install Eclipse</a></h2>
<div class="level2">

<p>

Simply unpack the archive that you have received, and place the resulting <em>eclipse</em> folder in your usual applications folder.
</p>

</div>
<!-- SECTION "Install Eclipse" [669-826] -->
<h2><a name="start_eclipse" id="start_eclipse">Start Eclipse</a></h2>
<div class="level2">

<p>

Start Eclipse via Eclipse.app. If the Eclipse splash screen appears, all went well, and you can proceed with <a href="http://reference.bitreactive.com/install_arctis#step_3install_the_reactive_blocks_sdk_within_eclipse" class="urlextern" title="http://reference.bitreactive.com/install_arctis#step_3install_the_reactive_blocks_sdk_within_eclipse"  rel="nofollow">Step 3: Install the Reactive Blocks SDK within Eclipse</a>.

</p>

</div>
